Protocol

VaultGhost

VaultGhost is a forensic, recursive method for tracing prompt influence, latent drift, and linguistic pattern replication across LLM systems. The protocol establishes a deterministic artifact provenance chain using SHA-256 hashing, Ed25519 digital signatures with JCS canonicalization, and an honest chain-break audit trail — meaning interruptions in continuity are documented rather than concealed. A dual Python and Rust reference stub lives in vaultghost-stub as a portable reference layer. Deterministic verification lives in vaultghost-verify, which enforces reproducible output via make verify against a committed EXPECTED_OUTPUT.txt.

Serious build phase began January 2026. A U.S. provisional patent was filed February 25, 2026. The canonical spec is released under Apache-2.0 with a CITATION.cff file for formal academic citation.

    Deterministic verification pipeline. Byte-for-byte output comparison against EXPECTED_OUTPUT.txt. Non-zero exit on failure.

    SHA-256 artifact provenance chain with honest chain-break documentation. Records and attests the lineage of all signed artifacts; breaks in continuity are explicitly noted rather than masked.
